# Carbon Fibre — Static Site

[![Deploy Pages](https://github.com/telsaleh/carbon-fibre/actions/workflows/deploy-pages.yml/badge.svg)](https://github.com/telsaleh/carbon-fibre/actions/workflows/deploy-pages.yml)

Static website for a carbon fibre installation business. Built with HTML5, CSS and minimal JavaScript. Intended for GitHub Pages hosting.

To preview locally open `index.html` in a browser or use a simple static server, e.g.:

```powershell
# from the repo root
python -m http.server 8000
# then open http://localhost:8000
```

Deployment to GitHub Pages:
- Create a repository on GitHub, push this code to the `main` branch.
- In GitHub repository settings, enable GitHub Pages to serve from the `main` branch root.
